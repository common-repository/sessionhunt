Hunting your daily web sessions in real time.

== Description ==
SessionHunt (https://sessionhunt.com) helps you monitor, analyze and collect the necessary information to learn how people are really using your app or website. Track visitor interactions on the web and mobile for free.

== Installation ==
= Installation =

To install SessionHunt, simply log in to your WordPress dashboard, navigate to the Plugins section and click Add New. In the search field type SessionHunt and click Search Plugins. Once you've found our plugin you can install it by clicking the Install Now¬ù button.

= Usage =

1. Open the settings page for SessionHunt (located in your sidebar).
2. Paste the embed code from your SessionHunt account.

== Frequently Asked Questions ==
= Where do I get my embed code? =
Simply head over to <a href="https://sessionhunt.com" target="_blank">SessionHunt</a>, login to your account and click on the settings icon -> embed code.
